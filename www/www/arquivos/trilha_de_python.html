Abrindo um arquivo
  1. A função OPEN é utilizado para realizar operações com arquivos em Python.
  2. Um arquivo pode ser aberto para leitura (read), escrita (write) ou acrescentar (append) conteúdo.
  3. Após utilizado, um arquivo sempre deve ser fechado.

Sintaxe da função open - modo leitura:

        arquivo = open("nomeDoArquivo.txt", "r")

        onde "nomeDoArquivo.txt" é o nome do arquivo que está sendo aberto
        e "r" indica que o arquivo está sendo aberto em modo de leitura somente.

        Esta função não lê o conteudo. Apenas abre o arquivo.

Sintaxe da função open - modo escrita (gravação):

        arquivo = open("nomeDoArquivo.txt", "w")

        onde "nomeDoArquivo.txt" é o nome do arquivo que está sendo aberto
        e "w" indica que o arquivo está sendo aberto em modo de escrita somente.

        Esta função não grava dados no arquivo. Apenas abre o arquivo.
Lendo os dados do arquivo
  1. Função read lê os dados como string
  2. Função readlines lê os dados como list

Sintaxe da função read:

        strConteudo = arquivo.read()

Sintaxe da função readlines:

        lstConteudo = arquivo.readlines()
